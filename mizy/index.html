miZy features

  • Orage-Pi-Zero - board based
  • fast boot + load system 5-10sec
  • spi flash - 8M-16M linux ready
  • mmc - tiny image size - 8M up
  • i2s - audio out
  • SPDIF - audio out
  • ssd1306 - i2c oled display
  • low power use 0.1A + low temp
  • miZy is full open source project
  • optimized linux kernel 3.4.113+
  • easy image builder on SPI | MMC | USB
  • easy full customizations for make your best embedded device
  • wifi + usb-ethernet + eth - http control
  • gpio buttons control
  • IR remote control
  • openwrt or simple busybox linux base
  • diskless mode, ramfs + squashfs + overlay system ready
  • easy rescue and safe mode + zero disk write mode ready
  • advaced U-boot + i2c display splash + spi boot & full linux load from spi
  • is cheep 7$ board + 3$ oled display + 1$ spi flash 16M, only 11$ for all
  • many other features
